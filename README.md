# Rick & Morty

Rick & morty es una aplicación creada con SwiftUI y está conectada a una API REST para mostrar personajes y detalles.
Incluye una pantalla de Splash para el inicio de la app, un Home dinámico para explorar, y un Detail con información única de cada personaje.

![Rickyandmorty](https://i.postimg.cc/sXvX0fmB/rickandmorty.png)

## Características

- **List:** Organiza vistas de manera ordenada en filas y columnas.
- **API REST:** La aplicación interactúa con una API REST para obtener información sobre los personajes y sus series.
- **MVVM:** Se ha utilizado el patrón de diseño MVVM para organizar el código, separando la lógica de negocio (Model), la representación de la interfaz de usuario (View) y el manejo de los datos (ViewModel).
- **Combine:** Implementación de Combine para manejar la reactividad de los datos.
- **Async/Await:** Uso de async/await para realizar solicitudes a la API de manera eficiente.
- **AppState:** Uso de estados en la aplicación para el manejo de errores.

## Uso

### Listado de personajes:
Al iniciar la aplicación, podrás ver una lista de personajes de Rick & Morty.

### Información de personajes:
Selecciona un personaje para ver la información detallada.
